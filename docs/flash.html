Flash Tool

Der ESP32-Code liegt im Ordner esp32/ElitaererUltimativerFutterautomat. Am einfachsten flashst du lokal mit PlatformIO oder Arduino IDE.

Variante A: PlatformIO

  1. ESP32 per USB anschliessen.
  2. Repository herunterladen oder klonen.
  3. Im Projektordner flashen.
  4. Nach dem Start mit dem Hotspot EU-Futterautomat-Setup verbinden.
  5. WLAN-Daten ueber http://192.168.4.1 eintragen.
pio run -t upload

Alternativ kannst du das Script tools/flash/flash_platformio.sh verwenden.

Variante B: Arduino IDE

  1. Arduino IDE installieren.
  2. Boardpaket esp32 by Espressif Systems installieren.
  3. Bibliotheken installieren: ArduinoJson, WebSockets, Adafruit NeoPixel.
  4. esp32/ElitaererUltimativerFutterautomat/ElitaererUltimativerFutterautomat.ino oeffnen.
  5. Board ESP32 Dev Module auswaehlen.
  6. Port auswaehlen und Upload starten.
  7. Nach dem Start mit dem Hotspot EU-Futterautomat-Setup verbinden.

Web-Flasher Hinweis

Ein echter Browser-Web-Flasher braucht eine vorher kompilierte .bin Firmware-Datei. Sobald eine Firmware-Binary erzeugt wurde, kann sie in docs/firmware/ abgelegt und mit ESP Web Tools verlinkt werden.

Bis dahin ist PlatformIO/Arduino IDE die zuverlaessigste Flash-Methode.
